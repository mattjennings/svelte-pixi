Renders a PIXI.Graphics. Use the `draw` prop to interact with the Graphics API.

<ComponentDescription name="Graphics" source="https://pixijs.download/dev/docs/scene.Graphics.html" />

## Usage

### Draw prop

The `draw` prop is a helper prop that gets called inside of an `$effect()`. This means it only
draws when the function itself or any state inside of it changes.

<SvelteTabs>
  <div slot="svelte-5">
    ```svelte live
    <script>
      import { Graphics, tick } from 'svelte-pixi'

      let size = tick(t => 100 + Math.sin(t.lastTime / 700) * 50) 

      function draw(graphics) {         
        graphics.clear()
        graphics
          .circle(0, 0, $size)
          .fill('darksalmon')
      }

    </script>

    <Graphics
      x={375}
      y={200}
      {draw}
    />
    ```
  </div>

  <div slot="svelte-4">
    ```svelte live
    <script>
      import { Graphics, tick } from 'svelte-pixi/svelte-4'

      let size = tick(t => 100 + Math.sin(t.lastTime / 700) * 50) 

      $: draw = (graphics) => {        
        graphics.clear()
        graphics
          .circle(0, 0, $size)
          .fill('darksalmon')
      }

    </script>
    
    <Graphics      
      x={375}
      y={200}
      draw={draw}
    />
    ```
    </div>
</SvelteTabs>
