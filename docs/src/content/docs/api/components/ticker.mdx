Children of this component can use [onTick](/docs/utilities/on-tick) to hook into the update loop.

## Usage

```svelte live
<script>
  import { Ticker, Text } from 'svelte-pixi'

  let lastDelta = $state(0)
  let frame = $state(0)
</script>

<Ticker
  ontick={(delta) => {
    frame++
    lastDelta = delta
  }}
>
  <Text 
    text={`Frame #: ${frame}\nLast Delta: ${lastDelta.toFixed(10)}ms`} 
    style={{ fill: 'white' }} 
    anchor={0.5} 
  />
</Ticker>
```
