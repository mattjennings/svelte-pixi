---
title: onTick
---

Child components of [Ticker](/docs/components/ticker) can use `onTick` to add a callback to the ticker. 
It will automatically remove itself on unmount.

```svelte live
<script>
  import { Text, onTick } from 'svelte-pixi'

  let frame = $state(0)
  let lastDelta = $state(0)

  onTick((delta) => {
    frame++
    lastDelta = delta
  })
</script>

<Text 
  text={`Frame #: ${frame}\nLast Delta: ${lastDelta.toFixed(10)}ms`} 
  style={{ fill: 'white' }} 
  anchor={0.5} 
/>
```

## Type Definition

```ts
function onTick(fn: (delta: number) => any, priority?: number | undefined): void
```
